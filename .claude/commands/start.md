# Старт проекта

Инициализация работы с проектом с изучением документации и контекста.

## Параметры
Параметры не требуются.

## Пример использования
/start

## Инструкции для Claude
При выполнении этой команды:

1. **Язык общения**: Отвечай на русском языке во всех последующих сообщениях.

2. **Изучение документации**: 
   - Просмотри все файлы в папке `_docs/` 
   - Ознакомься с содержимым документации
   - Запомни ключевые моменты архитектуры, API, требований

3. **Анализ проекта**:
   - Просмотри структуру проекта
   - Изучи основные файлы конфигурации (package.json, config files, etc.)
   - Ознакомься с исходным кодом для понимания текущего состояния
   - Определи этапы разработки и текущий прогресс

4. **Вывод результата**:
   - Кратко опиши структуру проекта
   - Перечисли основные компоненты/модули
   - Укажи текущий этап разработки
   - Отметь ключевые моменты из документации
   - Сообщи о готовности к работе

5. **Инструкция**
   - если у тебя есть альтернативное решение сначала обсуди его со мной
   - каждое решение начинается с обсуждения плана
   
Выполни все пункты последовательно и предоставь общий отчет о проекте.
