# Инструкции для разработки CS2 Marketplace

## 📁 Структура проекта

### Основные директории:
- `/app` - основной код приложения Laravel
  - `/Http/Controllers` - контроллеры (WebController, DashboardController, AuthController)
  - `/Models` - модели (User для админов, Client для пользователей маркетплейса)
  - `/Http/Middleware` - middleware (AuthenticateClient, IsBot)
- `/resources/views` - Blade шаблоны
  - `/layouts` - основные layout файлы
  - `/partials` - переиспользуемые компоненты (header, footer и т.д.)
  - `/dashboard` - страницы личного кабинета
- `/public` - публичные файлы (изображения, CSS, JS)
- `/routes/web.php` - маршруты приложения
- `/_docs` - документация проекта
- `/_template` - исходные HTML шаблоны для адаптации

## 🗄️ База данных

### Основные таблицы:
1. **users** - администраторы Filament панели
2. **clients** - пользователи маркетплейса (включая ботов)
   - Поля: name, email, steam_id, steam_avatar, steam_trade_url, balance, payment_password, is_verified, is_bot, locale

### Дополнительные таблицы (будут созданы):
- items - каталог предметов CS2
- listings - активные предложения на маркете
- trades - обмены
- transactions - финансовые операции
- bot_inventories - инвентари ботов
- price_history - история цен

## 🔐 Авторизация

- **Guard 'web'** - для админов Filament
- **Guard 'client'** - для пользователей маркетплейса
- Авторизация через Steam для клиентов
- Middleware: AuthenticateClient, IsBot

## 🚀 План MVP (Этап 1)

### Основной функционал:
1. **Авторизация через Steam**
2. **Быстрая продажа боту** - мгновенный выкуп по формуле
3. **P2P торговля** - через браузерное расширение
4. **Система ботов** - обычные пользователи с флагом is_bot
5. **Финансовая система** - пополнение/вывод средств
6. **Личный кабинет** - управление торговлей и настройками

### Ключевые особенности:
- Двуязычный интерфейс (ru/en) с самого начала
- Комиссия платформы 5% с продавца при P2P
- Минимальная цена предмета: 0.5₽
- Валидация предметов: >200 лотов на Steam маркете
- Таймаут трейда: 5 минут
