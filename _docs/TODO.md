# План разработки CS2 Marketplace - Страница маркетплейса

## Текущий этап: Разработка маркетплейса

### Фаза 1: База данных (1-2 дня)

#### 1.1 Создание таблиц
- [ ] **items** - каталог всех предметов CS2
  - steam_market_hash_name (уникальный идентификатор)
  - name_ru, name_en
  - type (пистолеты, ножи, перчатки и т.д.)
  - weapon (конкретное оружие: AK-47, AWP и т.д.)
  - rarity (Ширпотреб, Промышленное, Армейское и т.д.)
  - image_url
  - min_steam_price
  - steam_listings_count
  - is_valid (для быстрой продажи)

- [ ] **listings** - активные предложения
  - client_id (продавец)
  - item_id
  - price
  - float_value (износ)
  - pattern_index
  - status (active, sold, cancelled)
  - type (p2p, bot_sale)
  - steam_asset_id (для отслеживания конкретного предмета)

- [ ] **price_history** - история цен
  - item_id
  - source (steam, internal)
  - price
  - created_at

## Технические детали реализации:

### Производительность
- Кеширование популярных запросов
- Lazy loading изображений
- Debounce для поиска
- Оптимизированные SQL запросы

### Безопасность
- CSRF защита для всех форм
- Валидация всех входных данных
- Санитизация поисковых запросов
- Rate limiting для API

### Масштабируемость
- Подготовка к большому количеству товаров
- CDN для изображений
- Очереди для тяжелых операций
