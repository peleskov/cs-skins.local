## 🛠️ Команды для разработки

```bash
# Запуск проекта
composer run dev

# Синхронизация инвентаря пользователя
php artisan inventory:sync {client_id}

# Импорт предметов из wiki.cs.money
php artisan items:import {type}

# Создание тестового админа
php artisan db:seed --class=CreateAdminUserSeeder
```
