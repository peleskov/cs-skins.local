# TODO: Браузерное расширение CS2 Marketplace

## 📋 Обзор проекта

### Назначение
Браузерное расширение для автоматизации Steam трейдов на платформе CS2 Marketplace. Позволяет продавцам автоматически отправлять трейд-офферы покупателям без ручного вмешательства.

### Поддерживаемые браузеры
- **Chrome/Edge** (Chromium) - приоритет
- **Firefox** - вторая очередь
- **Safari** - будущие планы

## 🏗️ Архитектура расширения

### Компоненты
```
┌─────────────────┐    ┌──────────────────┐    ┌─────────────────┐
│   Background    │    │   Content Script │    │     Popup       │
│     Service     │◄──►│    (Steam)       │◄──►│      UI         │
│     Worker      │    │                  │    │                 │
└─────────────────┘    └──────────────────┘    └─────────────────┘
         ▲                        ▲                       ▲
         │                        │                       │
         ▼                        ▼                       ▼
┌─────────────────┐    ┌──────────────────┐    ┌─────────────────┐
│   Platform      │    │   Steam Web API  │    │   Local Storage │
│      API        │    │                  │    │                 │
└─────────────────┘    └──────────────────┘    └─────────────────┘
```
