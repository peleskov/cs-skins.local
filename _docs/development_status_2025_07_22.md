## 📋 Обзор проекта

CS:GO Skins Marketplace - это платформа для торговли скинами CS:GO с интеграцией Chrome расширения для автоматизации трейдов через Steam.

### Технологический стек:
- **Backend:** Laravel 12.x, PHP 8.3
- **Frontend:** Vue.js 3, Bootstrap 5
- **База данных:** MySQL
- **Очереди:** Laravel Horizon
- **Планировщик:** Laravel Scheduler
- **Расширение:** Chrome Extension (Manifest V3)

## 📝 Важные замечания

1. **Версия Laravel:** Используется Laravel 12.x, где планировщик задач настраивается в `routes/console.php`, а не в `Kernel.php`

2. **Статусы заказов:**
   - Order: `pending`, `paid`, `processing`, `completed`, `cancelled`, `refunded`
   - OrderItem: `reserved`, `trade_sent`, `completed`, `cancelled`
   - Trade: `pending`, `completed`, `cancelled`, `expired`

3. **Резервирование:** Товары резервируются на 5 минут. После истечения:
   - Статус листинга возвращается на `active`
   - Покупателю возвращаются средства
   - OrderItem помечается как `cancelled`
