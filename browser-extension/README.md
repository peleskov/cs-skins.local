# CS-SKINS.pro Trading Assistant - Browser Extension

Браузерное расширение для автоматизации Steam трейдов на платформе CS-SKINS.pro.

## 🚀 Быстрый старт

### Установка для разработки

1. **Клонируйте или скопируйте папку расширения**
   ```bash
   cd browser-extension/
   ```

2. **Загрузите в Chrome как unpacked extension**
   - Откройте Chrome и перейдите в `chrome://extensions/`
   - Включите "Developer mode" (Режим разработчика)
   - Нажмите "Load unpacked" (Загрузить распакованное)
   - Выберите папку `browser-extension/`

3. **Настройте расширение**
   - Кликните на иконку расширения в панели браузера
   - Войдите в свой профиль на CS-SKINS.pro
   - Скопируйте токен из раздела "Расширение" 
   - Вставьте токен в popup расширения

## 📁 Структура проекта

```
browser-extension/

    └── storage.js              # Утилиты локального хранилища
```

## 🔧 Технические детали

### Manifest v3 (Chrome/Edge)
- **Service Worker** вместо background scripts
- **Dynamic imports** для модульности
- **Content scripts** для интеграции с Steam
- **Host permissions** для доступа к Steam и платформе

### API Endpoints
Расширение взаимодействует с следующими API эндпоинтами:

- `POST /api/extension/auth` - Авторизация расширения
- `GET /api/extension/orders/pending` - Получение заказов
- `POST /api/extension/orders/{id}/trade-status` - Обновление статуса трейда
- `GET /api/extension/user` - Информация о пользователе
- `GET /api/extension/stats` - Статистика расширения

### Безопасность
- **Bearer токены** для авторизации
- **Rate limiting** на уровне API
- **CORS** настройки для безопасности
- **CSP** (Content Security Policy)

## 🎯 Основной функционал

### 1. Авторизация
- Связка расширения с аккаунтом через токен
- Проверка подлинности при каждом запросе
- Автоматическое обновление токенов

### 2. Мониторинг заказов
- Опрос новых заказов каждые 5 секунд
- Real-time уведомления о новых трейдах
- Очередь обработки заказов

### 3. Автоматизация Steam трейдов
- Извлечение Steam session данных
- Создание Trade Offers через Steam Web API
- Обработка ошибок и повторные попытки

### 4. Пользовательский интерфейс
- Статус подключения в реальном времени
- Статистика трейдов и заработка
- История последних операций

## 🔄 Жизненный цикл заказа

```
1. Покупатель оформляет заказ на сайте
   ↓
2. Расширение получает уведомление о новом заказе
   ↓
3. Content script создает Trade Offer в Steam
   ↓
4. Уведомление платформы об отправке трейда
   ↓
5. Покупатель принимает трейд в Steam
   ↓
6. Автоматическое завершение сделки
```

## 🛠️ Разработка

### Отладка
- Откройте DevTools для background script: `chrome://extensions/` → Details → Inspect views: background page
- Content script логи доступны в DevTools страниц Steam
- Popup логи в DevTools popup окна

### Тестирование
1. Создайте тестовый заказ на платформе
2. Убедитесь что расширение получает уведомление
3. Проверьте создание трейда в Steam
4. Проверьте обновление статуса на платформе

1. **Steam Guard Mobile**: Требуется подтверждение трейдов в мобильном приложении
2. **Rate Limits**: Ограничения Steam API на количество трейдов
3. **Session Timeout**: Необходимость переавторизации в Steam
4. **CORS**: Ограничения для cross-origin запросов

## 🐛 Troubleshooting

### Расширение не подключается
1. Проверьте правильность токена
2. Убедитесь что сайт доступен
3. Проверьте права доступа расширения

### Трейды не создаются
1. Убедитесь что вы авторизованы в Steam
2. Проверьте Trade URL покупателя
3. Убедитесь что предметы доступны для торговли

### Ошибки в консоли
- Откройте DevTools и проверьте ошибки
- Проверьте Network tab для API запросов
- Убедитесь что все файлы загружены
