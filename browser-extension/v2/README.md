# CS-SKINS Trading Assistant v2

Упрощенная версия браузерного расширения для работы с CS-SKINS.pro

## Особенности v2

- Минималистичный интерфейс с базовым функционалом
- Авторизация через Bearer token
- WebSocket соединение с сервером
- Ping-pong механизм для проверки активности
- Отправка Steam сессии по запросу сервера
- Поддержка светлой и темной темы
- Автоматическое определение Steam сессии

## Функционал

### UI
- Статусы подключения (Сервер/Steam)
- Информация о пользователе
- Одна кнопка включить/выключить

### WebSocket события
- `ping` / `pong` - проверка активности
- `request_session` - запрос Steam сессии
- `session_data` - отправка данных сессии

### Steam интеграция
- Извлечение cookies: sessionid, steamLoginSecure
- Определение steamID пользователя
- Автоматическое открытие Steam страницы при необходимости

## Установка

1. Перейдите в chrome://extensions/
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное"
4. Выберите папку browser-extension/v2/
5. Получите токен на сайте в профиле
6. Введите токен в расширении
